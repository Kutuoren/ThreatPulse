# ThreatPulse – Cyber Intelligence Dashboard

ThreatPulse is a lightweight, browser‑based cyber intelligence dashboard built with HTML, CSS, and JavaScript. It visualizes threat data in a clean, interactive interface and is designed for learning, demos, and rapid prototyping.

## Features
- Real‑time‑style threat visualization (static demo)
- Clean and responsive UI
- Multiple dashboard versions (`cyber_threat_dashboard.html`, `threatpulse_v2.html`)
- Easy to extend with real APIs or backend data

## How to Use
Open any of the HTML files in your browser:

- `cyber_threat_dashboard.html`
- `threatpulse_v2.html`

No installation or dependencies required.

## Project Structure
- `cyber_threat_dashboard.html` — first version of the dashboard  
- `threatpulse_v2.html` — improved and updated version  
- `assets/` — images, icons, styles (if included)  
- `*.pdf` — roadmap and prompt documents used during development
